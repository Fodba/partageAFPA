Pour démarrer ce DVD de formation :

Vous avez différentes possibilités pour démarrer ce DVD. Choisissez la méthode correspondant à votre système d’exploitation.

Tous systèmes d’exploitation : (Mac OS X, Windows, Linux)
Nom de fichier : start.html
Ce fichier démarre le DVD dans votre navigateur. Vous pouvez déplacer ou régler la fenêtre de votre navigateur à votre gré.

Windows
Nom de fichier : start.exe
Le DVD démarre dans une application autonome. Une fois que vous avez activé le démarrage automatique sur votre système, le fichier est exécuté. Avantage de cette méthode : il n’est pas nécessaire d’avoir un Flash Player sur votre système et vous pouvez visionner votre DVD de formation en mode plein écran.

Mac OS X
Nom de fichier : start.app
Si vous utilisez Mac OS X, vous pouvez démarrer la formation en exécutant ce fichier. Avantage de cette méthode : il n’est pas nécessaire d’avoir un Flash Player sur votre système et vous pouvez visionner votre DVD de formation en mode plein écran.

Linux
Il est également possible de démarrer le DVD sous Linux. Pour visionner un DVD de video2brain sous Linux, vous avez besoin d´une interface graphique de votre choix (par exemple, le navigateur web Mozilla) et du plug-in Flash pour Linux. Cependant, en raison des droits de licence, nous ne sommes pas autorisés à fournir ce plug-in sur le DVD.

Le Flash-Plugin pour Linux ainsi qu´une aide à l´installation peuvent être téléchargés depuis le site web d'Adobe. 

 

Vous pouvez lancer votre formation dans un navigateur web à l’aide du fichier start.html.

Vidéos pour les appareils mobiles :

Toutes les vidéos pour Smartphones et tablettes
Les fichiers vidéo au format MP4 sont destinés aux appareils mobiles sous iOS et Android, ainsi qu’aux appareils mobiles similaires capables de lire les fichiers MP4. Pour synchroniser les vidéos, les utilisateurs sous iOS doivent ouvrir la liste lecture du dossier Videos avec iTunes. Après avoir copié les vidéos dans iTunes, choisissez l’appareil souhaité et cliquez sur Synchroniser. Les utilisateurs sous Android peuvent copier les fichiers MP4 sur leur appareil et les consulter directement. Si le lecteur vidéo utilisé lit les listes de lecture au format M3U, les métadonnées peuvent également être importées.
